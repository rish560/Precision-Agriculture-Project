# FarmVerse – Precision Agriculture Platform

## Project Overview
FarmVerse is a web-based application designed to help users manage farms, crops, and agricultural activities efficiently.

## Team
Team D

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Spring Boot (Java)
- Database: MySQL
- Version Control: Git & GitHub

## Features
- User Login & Registration
- Dashboard
- Farm Management
- Crop Management
- Role-Based Access
